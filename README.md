# Projetos Python com Tkinter

Aplicações desktop desenvolvidas com Python e Tkinter, explorando criação de interfaces gráficas.

## Sobre

Este repositório contém projetos práticos de Python utilizando a biblioteca Tkinter para criação de interfaces gráficas desktop. Os projetos exploram componentes visuais como janelas, botões, menus e caixas de diálogo.

## Tecnologias

- Python 3
- Tkinter (GUI)

## Projetos

| Arquivo | Descrição |
|---|---|
| `Barra de opções.py` | Interface com barra de menu e opções |
| `Caixa de Opções Tkinter.py` | Componente de seleção com Tkinter |
| `testetkinter.py` | Testes e experimentos com Tkinter |
| `t1.py` | Primeiro projeto Tkinter |

## Como executar

```bash
python3 "Barra de opções.py"
```

> Requer Python 3 com Tkinter instalado (geralmente já incluso na instalação padrão do Python).
